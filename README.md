Для этого соревнования сделал более разжеванную демонстрацию использования duckdb - 0040preprocessing.ipynb. <s>Количество комментариев сопоставимо с количеством кода.</s> Количество комментариев значительно больше нуля. 

1) duckdb при нехватке ОЗУ использует ПЗУ (отличии от polars и pandas, ).
2) Где polar работает в один поток, duckdb молотит всеми доступными.

В результате, зная duckdb, невольно начинаешь смотреть с высока на жалобщиков на нехватку ОЗУ использующих polars и тем более pandas (как полярные медведи на панд).
